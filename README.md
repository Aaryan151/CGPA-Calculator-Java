# CGPA Calculator (Java)

A Java-based CGPA calculator that computes SGPA and CGPA using subject credits and grades.

## Features
- Uses arrays to store subject credits and grade points
- Calculates SGPA using weighted average
- Modular code with separate calculation method

## Technologies Used
- Java
- Arrays
- Loops
- Functions (Methods)

## How to Run
javac CGPACalculator.java
java CGPACalculator
